## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[

    ]}
>
<TabItem value="get">

Gets all properties from an individual <code>routing_rule</code>.
```sql
SELECT
region,
routing_rule_id,
routing_rule_arn,
domain_name_arn,
priority,
conditions,
actions
FROM awscc.apigatewayv2.routing_rules
WHERE region = 'us-east-1' AND Identifier = '{{ routing_rule_arn }}';
```
</TabItem>
<TabItem value="list">

Lists all <code>routing_rules</code> in a region.
```sql
SELECT
region,
routing_rule_arn
FROM awscc.apigatewayv2.routing_rules_list_only
WHERE region = 'us-east-1';
```
</TabItem>
</Tabs>

## `UPDATE` example

Use the following StackQL query and manifest file to update a <code>routing_rule</code> resource, using [__`stack-deploy`__](https://pypi.org/project/stack-deploy/).

```sql
/*+ update */
UPDATE awscc.apigatewayv2.routing_rules
SET PatchDocument = string('{{ {
    "Priority": priority,
    "Conditions": conditions,
    "Actions": actions
} | generate_patch_document }}')
WHERE region = '{{ region }}'
AND Identifier = '{{ routing_rule_arn }}';
```


## `DELETE` example

```sql
/*+ delete */
DELETE FROM awscc.apigatewayv2.routing_rules
WHERE Identifier = '{{ routing_rule_arn }}'
AND region = 'us-east-1';
```
