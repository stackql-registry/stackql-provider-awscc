<TabItem value="get">

Gets all properties from an individual <code>routing_rule</code>.
```sql
SELECT
region,
routing_rule_id,
routing_rule_arn,
domain_name_arn,
priority,
conditions,
actions
FROM awscc.apigatewayv2.routing_rules
WHERE region = 'us-east-1' AND data__Identifier = '<RoutingRuleArn>';
```
</TabItem>
<TabItem value="list">

Lists all <code>routing_rules</code> in a region.
```sql
SELECT
region,
routing_rule_arn
FROM awscc.apigatewayv2.routing_rules_list_only
WHERE region = 'us-east-1';
```
</TabItem>
</Tabs>

## `UPDATE` example

```sql
/*+ update */
UPDATE awscc.apigatewayv2.routing_rules
SET data__PatchDocument = string('{{ {
    "Priority": priority,
    "Conditions": conditions,
    "Actions": actions
} | generate_patch_document }}')
WHERE region = '{{ region }}'
AND data__Identifier = '<RoutingRuleArn>';
```


## `DELETE` example

```sql
/*+ delete */
DELETE FROM awscc.apigatewayv2.routing_rules
WHERE data__Identifier = '<RoutingRuleArn>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>routing_rules</code> resource, the following permissions are required:

### Create
```json
apigateway:CreateRoutingRule
```

### Read
```json
apigateway:GetRoutingRule
```

### Update
```json
apigateway:UpdateRoutingRule
```

### Delete
```json
apigateway:DeleteRoutingRule
```

### List
```json
apigateway:ListRoutingRules
```
