## `SELECT` examples

Gets all properties from an individual <code>module_version</code>.
```sql
SELECT
region,
arn,
description,
documentation_url,
module_name,
module_package,
is_default_version,
schema,
time_created,
version_id,
visibility
FROM awscc.cloudformation.module_versions
WHERE region = 'us-east-1' AND data__Identifier = '<Arn>';
```

## `DELETE` example

```sql
/*+ delete */
DELETE FROM awscc.cloudformation.module_versions
WHERE data__Identifier = '<Arn>'
AND region = 'us-east-1';
```

## Permissions

To operate on the <code>module_versions</code> resource, the following permissions are required:

### Create
```json
cloudformation:DescribeType,
cloudformation:DescribeTypeRegistration,
cloudformation:ListTypeVersions,
cloudformation:RegisterType,
s3:GetObject,
s3:ListBucket
```

### Read
```json
cloudformation:DescribeType
```

### Delete
```json
cloudformation:DeregisterType,
cloudformation:DescribeType
```
