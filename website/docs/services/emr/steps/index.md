## `SELECT` examples

Gets all properties from an individual <code>step</code>.
```sql
SELECT
region,
id,
action_on_failure,
hadoop_jar_step,
job_flow_id,
name
FROM awscc.emr.steps
WHERE region = 'us-east-1' AND Identifier = '{{ id }}';
```
