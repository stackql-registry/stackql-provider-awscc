## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[

        { label: 'list (identifiers only)', value: 'list' }
    ]}

<TabItem value="get">

Gets all properties from an individual <code>insight</code>.
```sql
SELECT
region,
insight_arn,
name,
filters,
group_by_attribute
FROM awscc.securityhub.insights
WHERE region = 'us-east-1' AND Identifier = '{{ insight_arn }}';
```
</TabItem>
<TabItem value="list">

Lists all <code>insights</code> in a region.
```sql
SELECT
region,
insight_arn
FROM awscc.securityhub.insights_list_only
WHERE region = 'us-east-1';
```
</TabItem>
</Tabs>

## `UPDATE` example

Use the following StackQL query and manifest file to update a <code>insight</code> resource, using [__`stack-deploy`__](https://pypi.org/project/stack-deploy/).

```sql
/*+ update */
UPDATE awscc.securityhub.insights
SET PatchDocument = string('{{ {
    "Name": name,
    "Filters": filters,
    "GroupByAttribute": group_by_attribute
} | generate_patch_document }}')
WHERE region = '{{ region }}'
AND Identifier = '{{ insight_arn }}';
```


## `DELETE` example

```sql
/*+ delete */
DELETE FROM awscc.securityhub.insights
WHERE Identifier = '{{ insight_arn }}'
AND region = 'us-east-1';
```
